# Orbital Launch Manifest Dark Editorial — Anchor Principles

- First viewport pairs a massive typographic promise with a sparse technical visual, leaving the product object quiet rather than over-explained.
- Narrative sections move from threshold/mission mood into machine anatomy, capability envelope, orbit/use-case cards, flight sequence, roadmap, and manifest form.
- Specs are shown as constrained labels, tables, and readouts; any unsupplied numbers must remain placeholders or be omitted.
- Responsive behavior collapses complex grids into single-column cards while preserving the section order and dark technical atmosphere.
