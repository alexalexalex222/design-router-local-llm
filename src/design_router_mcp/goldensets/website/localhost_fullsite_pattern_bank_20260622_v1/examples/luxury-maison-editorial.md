# Luxury Maison Editorial Serif

Use for luxury retail, boutique hospitality, fragrance, editorial brand houses, atelier, or high-touch product pages that need restraint, serif-led typography, ritual pacing, scarce CTAs, and a boutique footer/subscription flow.

## Reusable strengths
- serif_hierarchy
- editorial_pacing
- numbered_principles
- luxury_footer
- newsletter_form

## Pattern rules
- The page uses restraint as hierarchy: large serif type, long quiet paragraphs, thin dividers, and very few controls.
- Numbered principles create a slow scroll rhythm, followed by a high-contrast editorial quote band and a decisive CTA.
- Newsletter and footer sections are composed like brand literature, not conversion widgets.
- Mobile maintains the same literary order with simplified navigation and full-width controls.

## Anti-copy
- Do not reuse donor brand names, exact headlines, exact facts, exact product names, exact numbers, or exact visual diagrams.
- Use this example for layout grammar and interaction/state grammar only.
