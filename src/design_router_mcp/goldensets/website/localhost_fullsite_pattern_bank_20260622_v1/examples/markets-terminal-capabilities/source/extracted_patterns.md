# Markets Terminal Blue Dashboard — Support Pattern

- Hero copy is subordinate to the live product surface; the visual proof is the terminal itself, not a generic illustration.
- A ticker/status strip creates continuity between hero and content, then dense capability cards explain each panel’s job.
- Charts, order books, watchlists, and tables should use real supplied data or neutral placeholders clearly marked as sample.
- Mobile keeps table readability by stacking market groups and shortening surrounding prose, not by hiding the product surface.
