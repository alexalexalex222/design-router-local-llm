# Instrument Studio Minimal Interactive — Anti-Copy

- Use this pack as an abstract full-site pattern source, not as a template to clone.
- Do not copy donor brand names, product names, route names, exact headlines, section copy, prices, measurements, charts, form labels, code identifiers, or visual diagrams verbatim.
- Carry forward only reusable structure: page rhythm, content sequencing, component relationships, state design, responsive behavior, and proof discipline.
- Any metric, price, credential, benchmark, technical capability, certification, quote, or market value must come from the user's brief; otherwise omit it or mark it as a placeholder.
- Source screenshots are reference evidence only; do not embed their paths or pixels in generated pages.
