On variable type

        
A single outline, infinitely adjustable.

        

          
A variable font folds an entire family into one file. Instead of shipping separate Light, Regular and Black cuts, it stores the master outlines and the rules for interpolating between them along named axes — weight, width, optical size, slant and any custom dimension the designer chooses to expose.

          
The browser reads those axes from the font's fvar table and lets CSS address any point in the design space through font-variation-settings. Optical size is the quiet miracle: the same letter is drawn sturdier and more open when small, finer and higher-contrast when large, so it reads well at every size it is set.
