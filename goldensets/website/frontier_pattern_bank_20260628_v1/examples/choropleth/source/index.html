    Methodology. County and state boundaries are real, drawn from the public-domain us-atlas TopoJSON (US Census cartographic boundaries), pre-projected with Albers USA so Alaska and Hawaii sit as insets. The three indicators are synthetic: each county's value is generated by a deterministic seeded model (a smooth spatial gradient plus seeded per-county noise), normalised 0–100. These are not real statistics — they exist only to demonstrate the encoding. Color ramps use a 7-bin quantile scale; counties with no sample value render as a neutral hatch.
